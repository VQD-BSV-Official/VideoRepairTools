The goal is to learn & improve file repair techniques. Thanks!*

## 🖼️ Restore your video
How to fix:
- 🧩 Step 1: Extract H264 (IDR, P, I...Frame)
- 🛠️ Step 2: Replace the title (H264) with a different one
- 🔗 If other cases...
